# Meraki-Network-Automation-Suite
In environments with more than 200 branches, manually configuring devices is error-prone and consumes an excessive amount of operational time. These scripts were designed to standardize deployments and automate repetitive network and security tasks.
# Key FeaturesBulk Network Configuration:
Automation of network creation and configuration for new branch offices (VLANs, SSIDs, and routing policies).  Security Policy Enforcement: Scripts for the bulk deployment of firewall rules and uniform security policies across multiple MX68 devices.  
# Audit & Reporting:
Generation of automatic reports on connectivity status and policy compliance across the entire Meraki stack (MX, MS, MR). 
# Technologies Used:
Language: Python.  
API: Meraki Dashboard API (REST).  
Main libraries: requests, meraki.
